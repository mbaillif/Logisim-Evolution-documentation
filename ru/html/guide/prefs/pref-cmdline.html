Параметры командной строки

Вы можете изменять многие настройки приложения Logisim через параметры командной строки. Это может быть особенно полезным в лаборатории с одним компьютером на учащегося, где вы хотите запускать Logisim каждый раз одинаково, вне зависимости от того, как предыдущие студенты могли настроить программу.

Общий синтаксис командной строки выглядит следующим образом. 

java -jar имяJarФайла [параметры] [именаФайлов]

Необязательные дополнительные файлы, указанные в командной строке, будут открыты как отдельные окна в Logisim.

Следующий пример запускает Logisim в его базовой конфигурации. 

java -jar jarFileName --user-template plain -gates ansi -locale en
usage: Logisim-evolution [-b ] [--clear-prefs] [-f ] [-g ] [-h] [-l ] [-m ] [-n ] [--no-splash] [-o ] [-s ] [-t ] [--toplevel-circuit ] [-u ] [-v] [-w ]

Поддерживаемые параметры следующие.

-u, --user-template [empty|plain|filename]

Задаёт шаблон для использования в Logisim.
Использует стандартную модель (plain), пустую модель (empty), ппользовательскую модель по имени файла (filename).

-g, --gates [ansi|iec]

Задаёт какой использовать тип логических элементов.

-o, --locale идентификаторЯзыка

Задаёт какой перевод использовать. На момент написания поддерживаются следующие языки:

de Немецкий
el Греческий
en Английский
es Испанский
fr французский
ru Русский
it итальянский
nl голландский
jp Японский
pl Полиш
--clear-prefs

Очистить все настройки приложения при запуске, так что Logisim будет вести себя так, как будто он запущен в данной системе первый раз.

--no-splash

Скрывает стартовое окно заставки Logisim.

-h, --help

Отображает список параметров командной строки.

-v, --version

Отображает номер версии Logisim и выход.

-l, --load имя файла

Загружает изображение в оперативную память (работает только с -tty). Дополнительная информация в Другие опции верификации.

-s, --substitute file1 file2

Загрузите файл, заменив файл библиотеки (file1) на файл библиотеки (file2). Дополнительная информация в Библиотеки-заменители.

-t, --tty [table|speed|tty|halt|stats|binary|hex|csv|tabs]

Работает без графического интерфейса пользователя. Дополнительная информация в Другие опции верификации.

-w, --test-vector имя_схемы векторном профиле проектном профиле
Выполняет тесты имя_схемы в проектном профиле в функции векторов в векторном профиле. Дополнительная информация в Test Vector Window.
-m, --geometry [ШиринаxВысота|ШиринаxВысота+X+Y]

Открывает Logisim с окном, размер которого соответствует параметрам W (ширина) и H (высота) или с верхним левым углом, расположенным согласно параметрам X,Y.
. Пример: -m 600x400+100+100

-n, --new-file-format circ_input circ_output

Открыть файл .circ любой версии (circ_input) Logisim и записать новый (circ_output) в текущем формате, используемом в Logisim-evolution.
