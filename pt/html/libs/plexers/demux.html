Comportamento

Uma saída irá coincidir com a entrada na face oeste, se seu número coincidir com o valor atual recebido através da seleção de entrada na face sul. Para as outras saídas o valor será (0) ou flutuante (U), dependendo do valor do atributo Tri-state?. Se a selecção de entrada contiver bits não flutuantes (U) , todas as saídas serão flutuantes (U). Se a entrada seleccionada contém o erro de valor(E), então todas as saídas são erro(E). Acho útil pensar em um demultiplexador como análogo a um comutador de trilhos em uma estrada de ferro, controlado por uma seleção feita à entrada.

Pinos (componente supondo voltado para leste)

Na face oeste:
Entrada que recebe o valor a ser encaminhado para uma das saídas na face leste. Com largura de bits de acordo com o atributo Bits de Dados.
Na face leste:
Um número variável de saídas, são numeradas a partir de 0 começando na norte. O número de resultados é determinado por 2Bits Selecionados Com largura de bits de acordo com o atributo Bits de Dados.
Na face sul, lado esquerdo:
entrada Enable: Quando 0, todas as saídas consistem em todos os bits flutuantes (U), independentemente dos dados e das entradas seleccionadas. Largura de bits 1.
Na face sul, lado direito indicado por círculo cinzento:
Entrada selecionada: o valor dessa entrada determinará para qual a saída na face leste será encaminhado o valor recebido na face oeste. Com largura de bits de acordo com o atributo Bits para selecção

Atributos

Quando o componente for selecionado ou estiver sendo acrescentado os dígitos de 1 a 5 poderão ter o seu atributo Bits para selecção alterado, Alt-0 até Alt-9 irão alterar o seu atributo Bits de dados, e as teclas com setas poderão alterar o seu atributo Posição.

Posição
A Posição do componente (especificará qual dos lados terá as saídas).
Selectionar posiçâo
A localização das linhas de selecção e activação em relação ao componente.
Bits para selecção
A largura em bits da entrada selecionada do componente na face sul. O número de saídas para o demultiplexador será 2 Bits Selecionados.
Bits de dados
Largura dos bits de dados que serão roteados pelo demultiplexador.
Tri-state?
Especificará se as saídas não selecionadas deverão ser flutuantes (Sim) ou zero (Não).
Desabilitar saida
Especificará qual deverá ser o valor de cada bit das saídas quando o componente estiver desabilitado (ou seja, quando o pino de habilitação estiver em 0). Opções incluem zero (0) e flutuante (U); e neste caso, as saídas estarão efetivamente desconectadas de quaisquer outras portas.
Incluir Enable?
O componente tem uma entrada de activação quando este atributo é Sim Este atributo é utilizado principalmente para suportar circuitos construídos usando versões mais antigas de Logisim que não forneciam uma entrada de activação.
