Installing Logisim-evolution standard

1 - Download version
You can download the .msi installer from : github
2 - launch the installer
On first launch, Windows will display a security screen, click on the additional information link and then on the Run anyway button. then click on the Next button.
3 - Specify installation file
The default installation folder is C:\Program Fileslogisim-evolution\. If you're happy with it, click Next, then Install.
4 - Installation complete
Click on Finish. You should find a shortcut on the desktop and logisim-evolution.exe in the installation folder. The default is C:\Program Fileslogisim-evolution\.
