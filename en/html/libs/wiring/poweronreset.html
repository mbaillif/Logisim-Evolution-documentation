Power on Reset

Library: Wiring
Introduced: 2.14.6
Appearance:

Behavior

The POR component emits a signal according to the Port transition attribute at simulation startup for a period defined by the Reset Time attribute. This happens either when the project is opened by logisim or when you use the Restart simulation menu or using the Pook tool().
In step-by-step mode the transition will take place after the delay but in synchronization with the pressing of the key (Ctrl-I).

Pins

This component has only one one-bit pin whose value will represent the current state of the reset. The location of this pin is specified in the attribute Facing.

Attributes

When the component is selected or added, the arrow keys change its Facing attribute.

Facing
The side of the drawn component where its output pin should be located.
Draw Size
Select different display dimensions for this component.
Port transistion
Sets the transition (up to down, down to up) after the time set by the Time reset property.
POR-Time
The time in seconds before the signal goes down to the low level (0).
