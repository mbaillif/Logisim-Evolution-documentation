Programmable logic array

Library: Gates
Introduced: 3.2
Appearance:

Behavior

The Programmable Logic Array (PLA) will present on its output the result of a combinatorial logic function. If one of the inputs is in error (E) or undefined (U), it will consider that all its inputs are at 1. This component resembles a ROM, and you need to enter the programming in a very similar way. Programming can be exported and imported from a file. In the Input/Output library (extra), there's a PLA coposant that better represents the internal workings of these components.

Pins (assuming component faces east)

West edge:
The component input, Bit width matches Bit width in attribute.
East edge:
The component output, Bit width matches Bit width out attribute.

Attributes

When the component is selected or being added, the arrow keys alter its Facing attribute.

Facing
The direction of the component, its output relative to input.
Bit width in
The bit width of the component's inputs.
Bit width out
The bit width of the component's output.
Program
Opens the programming window.
Label
The text within the label associated with the gate.
Label Font
The font with which to render the label.

Poke Tool Behavior

None.

None.
