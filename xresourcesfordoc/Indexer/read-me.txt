In this folder you'll find the jsearch-indexer.jar program.It is used to build keyword search indexes for logisim's Javahelp.Usage:   java JavaHelp.Index optionsOptions  -locale language_country_variant         If specified participates in the construction         from the documents path         and the name of the db         Ex : 'en/html/'  and 'search_lookup_en'  -helppath root path of help files         by default the launch path  -docpath  document path         by default 'helppath' + html/  -db file  generated database file name         by default 'search_lookup'  -stoptxt  Stop text file name         by default in helpppath 'jhindexer-stops.txt'  -verbose  verbose documentation  -nostopwords ignore stop words  -logfile  log file name  -c file   config file name
