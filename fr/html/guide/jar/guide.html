En savoir plus

Au-delà de la séquence d'exemples fournie ici, le code source de Logisim fournit de nombreux exemples supplémentaires, bien qu'ils n'illustrent pas toujours la même attention portée à la lisibilité et à la bonne conception.

Pour une portabilité maximale dans les versions futures, vous devriez vous en tenir autant que possible aux classes des paquets instance ...instance, ...data, et..tools . Bien sûr, vous pouvez utiliser les API d'autres paquets, mais ils sont plus vulnérables aux changements dans les futures versions de Logisim.

Vous pouvez pauser vos questions ou proposer des suggestions d'améloration sur le site de Logisim-evolution. Vos commentaires seront toujours les biens venus.

Distribution

Vous êtes libre de distribuer sans restriction les JAR que vous développez. Les restrictions de la GPL s'appliquent toutefois si des parties de votre travail sont dérivées de parties du code source de Logisim (publié sous la GPL). La dérivation du code d'exemple dans cette section du Guide de l'utilisateur n'entraîne pas de telles restrictions ; ces exemples sont publiés sous la licence MIT.

Prenez contact avec les administrateurs du site de Logisim-evolution si vous désirez partager vos bibliothèques avec d'autres utilisateurs ou aider au développement de ce logiciel
