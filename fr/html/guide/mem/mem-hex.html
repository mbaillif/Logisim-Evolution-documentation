Editeur hexadécimal

Logisim inclut un éditeur hexadécimal intégré pour visualiser et éditer le contenu des mémoires. Pour y accéder, ouvrez le menu contextuel du composant (clic-doit sur le composant) et sélectionnez | Modifier le contenu |. Pour les ROM il est aussi possible dans les propriétés de sélectionner Contenu.

Les chiffres en italique dans la colonne de gauche affichent les adresses mémoire en hexadécimal. Les autres nombres affichent la valeur des cellules à partir de cette adresse mémoire. L'éditeur hexadécimal peut afficher quatre, huit ou seize valeurs par ligne, en fonction la largeur de la fenêtre. Pour faciliter la visualisation et la lecture, ils sont regroupés par quatre avec un espace entre les groupes.

Il est possible de naviguer dans la mémoire à l'aide de la barre de défilement ou du clavier. La saisie de caractères hexadécimaux modifiera la valeur de la cellule sélectionnée.

  • Les touches Flèche bas ou Entrée feront défiler d'une ligne vers le bas.
  • Les touches Flèche haut ou Ctrl+Entrée feront défiler d'une ligne vers le haut.
  • La molette de souris permet ce même défilement.
  • Les touches Flèche droite ou Espace feront défiler d'une case à droite.
  • Les touches Flèche gauche ou Retour arrière feront défiler d'une case à gauche.
  • Les touches Page bas ou Ctrl+Espace feront défiler d'une page vers le bas.
  • Les touches Page haut ou Ctr+Retour arrière feront défiler d'une page vers le haut.
  • Les touches Home et End pemetent de déplacer le curseur au début ou a la fin du document.
  • La touche Del réinitialise la cellule sélectionnée avec un zéros.

Vous pouvez sélectionner une plage de valeurs en faisant clic-droit et glisser la souris ou en naviguant dans la mémoire avec le clavier tout en appuyant sur la touche Maj.
Vous pouvez copier, couper, coller, sélectionner les données de la mémoire depuis le menu | éditer | ou avec les raccourcis associés Ctrl-C, Ctrl-V, Ctrl-X…
Le presse-papiers peut également être transféré vers/depuis d'autres applications.

Deux boutons (Ouvrir…, Enregistrer…) permettent de charger ou d'enregistrer les données depuis un fichier. Vous trouverez les information sur les types de fichier dans Menu contextuel et fichiers
