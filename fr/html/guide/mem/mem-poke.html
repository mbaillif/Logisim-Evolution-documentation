Editer la mémoire

Vous pouvez manipuler le contenu de la mémoire en utilisant l'outil Pousser (), mais l'interface pour cela est sévèrement limitée par des contraintes d'espace. Par ailleurs il existe un éditeur hexadécimal intégré bien plus pratique.

Néanmoins, pour afficher et modifier les valeurs dans le circuit, l'outil Poke dispose de deux modes de fonctionnement: vous pouvez modifier l'adresse affichée et vous pouvez modifier une valeur individuelle.

Sélection de l'adresse

Pour sélectionner une adresse, cliquez à l'extérieur du rectangle d'affichage ou sur la colonne d'adresse. Logisim dessinera un rectangle rouge autour de l'adresse supérieure.
Pour quitter l'éditeur, simplement cliquez ailleurs sur le schéma.

  • Si vous entrez une valeur hexadécimale, l'éditeur se positionnera à l'adresse demandée.
  • Les touches Flèche bas ou Entrée feront défiler d'une ligne vers le bas.
  • Les touches Flèche haut ou Ctrl+Entrée feront défiler d'une ligne vers le haut.
  • La molette de souris permet ces mêmes défilements.
  • Les touches Page bas ou Ctrl+Espace ou Espace feront défiler d'une page vers le bas.
  • Les touches Page haut ou Retour arrière Ctrl + Retour arrière feront défiler d'une page vers le haut.

Modification des données

Pour modifier les données, cliquez sur une des cases de donnée. Logisim dessinera un rectangle rouge autour de la case.

Vous pouvez ensuite saisir la valeur de la cellule mémoire sélectionné en hexadécimale.

  • Les touches Flèche bas ou Entrée feront défiler d'une ligne vers le bas.
  • Les touches Flèche haut ou Ctrl+Entrée feront défiler d'une ligne vers le haut.
  • La molette de souris permet ce même défilement.
  • Les touches Flèche droite ou Espace feront défiler d'une case à droite.
  • Les touches Flèche gauche ou Retour arrière feront défiler d'une case à gauche.
  • Les touches Page bas ou Ctrl+Espace feront défiler d'une page vers le bas.
  • Les touches Page haut ou Ctr+Retour arrière feront défiler d'une page vers le haut.
  • La touche R réinitialise l'ensemble de la mémoire avec des zéro ou aléatoirement selon les paramètres du projet…
  • La touche Del réinitialise la cellule sélectionnée avec un zéros.

Pour sortir de l'éditeur, simplement cliquez ailleurs sur le schéma.
