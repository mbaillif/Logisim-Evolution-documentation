Installation de Logisim-evolution standard

1 - Téléchargement de la version
Vous pouvez télécharger le l'installer .msi obtenu depuis github : github
2 - lancez l'installer
Au premier lancement, Windows affichera un écran de sécurité, cliquez sur le lien Information complémentaires et ensuite sur le bouton Executer quand même. Le logiciel est en anglais, mais cela n'a pas d'importance cliquez sur le bouton Next
3 - Précisez le dossier d'installation
Le dossier d'installation par défaut est C:\Program Files\logisim-evolution\. S'il vous convient, cliquez sur le bouton Next puis sur Install
4 - L'installation est terminée
Cliquez sur le bouton Finish. Vous devriez trouver un raccourci sur le bureau et le logiciel logisim-evolution.exe dans le dossier d'installation. Par défaut c'est C:\Program Files\logisim-evolution\
