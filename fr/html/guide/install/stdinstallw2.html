Installer rapidement une autre versions standard sous windows

1 - Téléchargement de la version
Vous pouvez télécharger télécharger le fichier .jar depuis github : github
2 - Sauvegardez l'installation précédente
Dupliquer le dossier de l'installation précédente (par défaut :C:\Program Files\logisim-evolution\ ) et nommez le avec la version que vous désirez installer par exemple C:\Program Files\logisim-evolution39\
2 - Déposez le fichier .jar
Déposez le fichier .jar dans le sous-dossier .\app\ du nouveau dossier d'installation.
3 - Modifiez le fichier de configuration
Modifiez les lignes suivantes du fichier de configuration :
- app.classpath=nouveau nom du .jar
- java-options=la version du fichier

par exemple vous auriez pour la version 3.9 de dev ceci :
   ===========
  [Application]
  app.classpath=$APPDIR\logisim-evolution-3.9.0dev-all.jar
  app.mainclass=com.cburch.logisim.Main

  [JavaOptions]
  java-options=-Djpackage.app-version=3.9.0
  ===========
5 - Créez un raccourci
Allez dans le nouveau dossier d'installation et exécutez un clic-droit sur le logiciel logisim-evolution.exe et sélectionner le sous-menu Crée un raccourci.
5 - Tester
Cliqze sur votre raccourci et dans le menu aide sous-menu a propos vous trouverez l'information sur la version qui s'exécute.
